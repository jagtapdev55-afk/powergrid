Smart Meter Integration

When a real smart meter is installed — this is the only code it needs:

# Runs inside the smart meter every 15 minutes automatically

reading = meter.get_current_reading()

response = http.post("{{ base_url }}meter-reading/submit/", {
    "consumer_number":  meter.consumer_number,
    "current_reading":   reading.value,
    "previous_reading":  meter.last_reading,
    "reading_date":      today(),
    "device_id":         meter.device_id,
    "voltage":           meter.voltage,
    "current_amp":       meter.current,
    "power_factor":      meter.power_factor,
    "signal_strength":   meter.signal,
    "tamper_alert":      meter.tamper_detected,
    "source":            "smart_meter",
    "api_key":           "{{ api_key }}"
})

# Done. Data is in your system instantly.

Response Codes
